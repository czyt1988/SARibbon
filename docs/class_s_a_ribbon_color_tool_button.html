enum SARibbonColorToolButton::ColorStyle

颜色样式

枚举值
ColorUnderIcon 

颜色在icon下方，这个要求必须设置icon

ColorFillToIcon 

颜色作为icon，这个模式下在setColor会自动生成一个颜色icon替换掉原来的icon，因此setIcon函数没有作用
