◆ setTextEllipsisAspectFactor()

void SARibbonToolButton::setTextEllipsisAspectFactor ( qreal fac = 1.4)
static

文本宽度估算时的宽度比高度系数

参数
fac系数，默认为1.4，此系数越大，按钮允许的宽度越宽

超过此系数的宽度时，开始尝试换行，例如按钮高度为h，如果单行文本的宽度大于h*系数，则按钮将不进行横向拉伸，类似于maxwidth效果

此系数和maxwidth取最小值
