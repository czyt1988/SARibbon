Ribbon控件中的分类选项卡页面（Category）

每个Category代表Ribbon中的一个标签页，包含多个面板(Pannel)

◆ removePannel() [2/2]

bool SARibbonCategory::removePannel ( SARibbonPannel * pannel)

移除Pannel，Category会直接回收SARibbonPannel内存

参数
pannel需要移除的pannel
注解
移除后pannel为野指针，一般操作完建议把pannel指针设置为nullptr

此操作等同于：

SARibbonPannel* pannel;
...
category->takePannel(pannel);
pannel->hide();
pannel->deleteLater();
